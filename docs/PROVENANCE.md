# Provenance

Sentinel Combination is a new independent codebase informed by:

- Sentinel Chain commit `6b8af3675cc4ecf6cea879f740d614764805756c`
- Sentinel Iron commit `f26c1c2010f1c2e94a13147f58d6dd99c6c9bb21`

Selected Chain concepts include signal normalization, bracket plans, staged exits, trailing behavior, break-even and profit locks, and operator-facing trade management.

Selected Iron concepts include order lifecycle, fill idempotency, readiness, reconciliation, margin discipline, instrument rules, phase-based reversals, kill switch, cancellation, emergency flattening, and broker-authoritative accounting.

The source repositories are not runtime dependencies. Their histories remain independent and may diverge.
